Комбо: Context + Router + localStorage
Проект: избранное

Список рецептов; у каждого кнопка «В избранное».
Context хранит массив id избранного.
Страница «Избранное» (/favorites) читает из контекста и показывает только избранные.
При изменении избранного сохранять массив id в localStorage и при старте приложения подставлять в контекст.
